Personal portfolio site built with React 18, TypeScript, Vite, Tailwind CSS, and shadcn/ui.

## Setup

```bash
npm install
npm run dev
```

Dev server runs on `http://localhost:8080`.

## Build

```bash
npm run build
npm run preview
```

## Tests

```bash
npm test
```

## Stack

- **Framework:** React 18 + TypeScript
- **Build:** Vite (SWC)
- **Styling:** Tailwind CSS v3, tailwindcss-animate
- **Components:** shadcn/ui (Radix primitives)
- **Routing:** React Router v6
- **Data fetching:** TanStack React Query (planned for GitHub API)
- **Font:** Ferro Rosso (custom)
